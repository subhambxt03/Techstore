-Tech Store – Full-Stack E-Commerce Web Application

-Live Demo: https://techstore-1-78ms.onrender.com/


-Overview

      Full-stack e-commerce web application built to demonstrate real-world frontend and backend integration.
      Focuses on dynamic data handling, REST APIs, and clean UI design.


-Tech Stack

      Frontend: HTML, CSS, JavaScript
      Backend: Flask (Python)
      Database: SQLite
      APIs: RESTful APIs (JSON)
      Deployment: Render


-Features

      Dynamic product listing loaded from the backend
      Category-based product browsing
      Responsive and clean user interface
      Asynchronous frontend–backend communication using Fetch API
      JSON-based data exchange
      Modular backend routing with Flask
      Lightweight and efficient database using SQLite


-Backend Functionality

       REST API endpoints to serve product data
       Server-side logic handled using Flask
       Database operations for storing and retrieving products
       Proper separation of frontend and backend responsibilities


-Frontend Functionality

       Dynamic UI rendering without page reloads
       Structured layout with reusable components
       Clean navigation and product grid design
       Scalable frontend architecture for future enhancements


-Learning Outcomes

       Practical understanding of full-stack web development
       Hands-on experience with REST APIs and JSON
       Improved debugging and problem-solving skills
       Experience deploying a Flask application to production
